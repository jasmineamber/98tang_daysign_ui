# 环境变量
| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| SEHUATANG_HOST | 域名（不带 https://） | sehuatang.org |
| SEHUATANG_FID | 板块 ID | 103 |
| SEHUATANG_ACCOUNTS_NUM | 签到用户数量 | 1 |
| SEHUATANG_REPLY_TIMES | 回复次数 | 1 |
| SEHUATANG_CHROME_PATH | 浏览器路径 | /usr/bin/google-chrome |
| SEHUATANG_FETCH_COOKIES_1 | 第一个账号 Cookies（多个账号修改结尾的数字） | — |
| BARK | BARK通知地址 | — |
| BARK_ICON | BARK通知图标 | — |
